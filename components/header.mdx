## Overview

The `Header` component is a simple, stateless React component that displays the application title at the top of the page. It provides consistent branding across the CO2 monitoring application.

## Component Code

The complete component is located at `src/components/Header.js`:

```javascript
import React from "react";
import './Header.css';

function Header() {
    return (
        <header>
            <h1 className="titulo">Sistema de Monitoreo de CO2</h1>
        </header>
    );
}

export default Header;
```

## Usage

The Header component is imported and rendered at the top of the main App component:

```javascript
import React from "react";
import Header from "./components/Header";
import CO2Dashboard from "./components/CO2Dashboard";
import "./styles.css";

function App() {
  return (
    <div>
      <Header />
      <CO2Dashboard />
    </div>
  );
}

export default App;
```

<Note>
The Header is rendered before the CO2Dashboard component to ensure it appears at the top of the page.
</Note>

## Styling Approach

The component uses a separate CSS file (`Header.css`) for styling. This follows React best practices for component-level styling:

- **Separation of Concerns**: Keeps JSX focused on structure while CSS handles presentation
- **Maintainability**: Makes it easy to update styles without modifying component logic
- **Reusability**: The `titulo` class can be referenced or extended in other stylesheets

<Tip>
To customize the header appearance, modify the styles in `src/components/Header.css`. Common customizations include changing font size, color, background, padding, and text alignment.
</Tip>

## Component Props

The Header component does not accept any props. It always renders the same content.
